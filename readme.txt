C# PokeBattle Console App

My app is a very basic Pokemon battle simulator between yourself and one other trainer. 
Both your teams are stored in Lists and then randomly selected. The value of Attack is then compared to Health's value and 
if it is higher, you win!

In order to prepare myself for building an app in a language I hadn't used before, I utilised a number of resources I have 
documented below: 

https://docs.microsoft.com/en-us/visualstudio/get-started/csharp/tutorial-console?view=vs-2022 - a brief tutorial on how to 
code a simple Calculator app in C#

https://www.w3schools.com/cs/index.php
